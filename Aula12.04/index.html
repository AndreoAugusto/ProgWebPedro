HTML Avançado e Seletores CSS

O objetivo hoje é aprender conceitos avançados de HTML e utilizar seletores CSS

Fundamentos de HTML5 e CSS: Estrutura, Estilo e Interatividade

Tags semânticas no HTML5

As tags semânticas, como header, footer, article e section, conferem significado ao conteúdo, melhorando a acessibilidade e a indexação nos motores de busca.

Tabelas e listas

As tabelas são criadas com table, tr, td e th, enquanto as listas podem ser ordenadas (ol) ou não ordenadas (ul), utilizando li para cada item.

Formatação de texto e cores

A formatação de texto é realizada com tags como para negrito e para itálico. As cores são aplicadas via CSS, utilizando propriedades como color, background-color e font-size.

Introdução aos seletores CSS

Seletores CSS permitem aplicar estilos a elementos HTML específicos. Podem ser básicos (por tag, classe ou ID), combinadores (como descendentes ou filhos diretos) e pseudo-classes (como :hover).

Agrupamento e herança de estilos

O agrupamento de seletores CSS permite aplicar um conjunto comum de estilos a múltiplos elementos. A herança permite que elementos filhos recebam estilos de seus elementos pais, facilitando a manutenção e consistência do design.
